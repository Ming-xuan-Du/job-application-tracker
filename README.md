# Job Application Tracker

A full-stack web application for tracking job and internship applications.

## Tech Stack

- Frontend: React, TypeScript, Tailwind CSS
- Backend: FastAPI, Python
- Database: SQLite

## Planned Features

- Add job applications
- View all applications
- Update application status
- Delete applications
- Search and filter applications
- View application statistics

## Project Goal

This project is built to practice full-stack development and create a resume-ready software engineering project.
